Data-Commons

The Data Commons Community

Data commons community is a group of data engineers and data scientists. Our goal is to provide a collection of rich, high-performance libraries to process data at scale. We are currently building these tools to work with Apache Spark.

Products

A set of libraries which are designed for data cleaning, quality checks and providing useful insights to data. Our core project prep-buddy which a library for cleaning, transforming and executing other preparation tasks for large datasets on Apache Spark.
